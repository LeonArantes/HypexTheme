> Open your Terminal
```
    $ git clone https://github.com/LeonArantes/HypexTheme.git ~/.vscode/extensions/HypexTheme
```

### Installation Extensions Visual Studio Code
> Launch VS Code Quick Open (Ctrl+P)
```
    ext install Hypex.Hypex
```
